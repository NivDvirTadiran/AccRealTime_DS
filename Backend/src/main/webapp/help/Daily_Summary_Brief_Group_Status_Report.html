Daily Summary Brief Group Status Report

The Daily Summary Brief Group Status Report displays brief information about one or more agent groups. This report presents the daily data accumulated since midnight.

To create a Daily Summary Brief Group Status window:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select Groups from the Template tab.

  1. Select the Daily Brief Report template. The Select Multiple Group dialog box is displayed.

  2. Select the groups for which you want to obtain agent group status information, and then click Select.

Note: You can change the groups presented in the report by selecting the current groups in the top bar of the window, and then selecting the required groups from the Select Multiple Group dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

The following columns are displayed in the Daily Summary Brief Group Status Report:

Column

Description

Group Name

The name of the agent group.

Group No.

The number assigned to the group.

Above T.ASA

The number of calls currently in the queue, for which the target Average Speed of Answer was exceeded.

Calls in Q

The number of calls currently in the queue.

Max in Q

The longest time of a call currently waiting in the queue for the group.

No. Agns

The number of agents which are currently logged in.

No. Release

The number of agents in the group that are in the Release state.

No. Idle

The number of agents in the group that are available.

No. ACD

The number of agents currently speaking on ACD calls.

Calls ACD

The number of ACD calls handled so far, since last midnight.

TSF

The Target Service Factor. The TSF indicates how well the group is providing the level of service defined for it, via the Target ASA (Average Speed of Answer).

Note: The TSF will be calculated according to the method selected in the Aeonix Contact Center Administration.

Calls Abnd

The number of calls in the group that were abandoned before being answered.

Calls Intrfl

The number of calls that were re-routed to another destination due to call interflow.

 

Note:

You can adjust the width of the columns by dragging the bottom right hand corner of the window.
