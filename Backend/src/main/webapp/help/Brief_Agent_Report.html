Brief Agent Report

The Brief Agent Report shows the current activity of each agent in the report, how long the agent has been engaged in that activity, and the login status of the agent.

To create a Brief Agent Report window:

  1. Click  (Open Report or Template) in the Control Area.

  2. In the Open dialog box, select Agents from the Template tab.

  3. Select the Brief Report template. The Select Multiple Agent dialog box is displayed.

  4. Select the agents for which you want to obtain agent status information, and then click Select.

Note: You can change the agents presented in the report by selecting the current agents in the top bar of the window, and then selecting the required agents from the Select Multiple Agent dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

The Brief Agent Report contains the following columns:

Column

Description

Agent Name

The name of the agent.

Agent No.

The ID number entered by the agent on login. This number is used to identify the agent in the system.

Agent Exten.

The internal extension that the agent is currently logged in on.

State

The agent’s state. This can be one of the following:

Idle

No calls are being handled and the agent is free to receive calls.

Release

The state of the agent when the Release button is activated. In this state the agent or station is still part of the ACD, but not available to receive any new ACD calls. The station can still receive incoming non-ACD calls in this state.

Ring

An ACD call is ringing at an agent’s extension.

Wrap Up

The agent has completed a call and is now in the wrap-up state.

ACD

The agent is currently talking on an ACD call.

Busy

The agent station is busy ringing an incoming non-ACD call or dialing an outgoing non-ACD call.

Inc

The agent answered an incoming, non-ACD call.

Out

The agent is making an outgoing, non-ACD call.

Out+Rls

The agent is making an outgoing outside line call, while the station is in Release state.

Inc+Rls

The agent received an incoming call (i.e., non-ACD call), while the station is in Release state.

Bsy+Rls

The agent is ringing an incoming non-ACD call or dialing an outgoing non-ACD call, while the station is in Release state.

Logout

The agent is NOT logged into the system

OACD

The agent is currently on an outbound ACD call

O-Rsrv

The agent is reserved for an outbound ACD call

State Time

The time the agent station is in the state indicated in the State column.

Note 1: When the agent completes a non-ACD activity while in Release state, the State Time continues to count the Release time from the time the agent first entered the Release state.

Note 2: If, when in Idle state, the agent’s extension becomes busy with a non-ACD call and then returns to Idle state, the State time continues to count the Idle time from the last ACD call the agent completed. 

Release Code

This column shows, for every released agent, the name of the release code selected by the agent. Note that there are two system defined release codes. The first is ‘No Code’ and the second is ‘Forced Released’.

ACD Calls

This column shows the number of incoming ACD Voice calls that the agent is currently handling.

O.ACD Calls

This column shows the number of outbound ACD Voice calls that the agent is currently handling.

Chat Contacts

This column shows the number of incoming ACD chat contacts that the agent is currently handling.

Email Contacts

This column shows the number of incoming ACD email contacts that the agent is currently handling.

ACD Calls Answered

This column shows the number of incoming ACD Voice calls that the agent answered since the last login to the system.

O.ACD Calls Answered

This column shows the number of outbound ACD Voice calls that the agent answered since the last login to the system.

Chat Contacts Answered

This column shows the number of incoming ACD chat contacts that the agent answered since the last login to the system.

Email Contacts Answered

This column shows the number of incoming ACD email contacts that the agent answered since the last login to the system.

N.ACD Calls Answered

This column shows the number of non ACD calls (incoming and outgoing, internal or external) that the agent answered since the last login to the system

DNIS

The DNIS value that arrived with the current call of the agent.

ANI

The ANI value that arrived with the current call of the agent.

Current ACD Group

The number of the ACD group the agent is working on in the current ACD call.

 

Note:

You can adjust the width of the columns by dragging the bottom right hand corner of the window.
