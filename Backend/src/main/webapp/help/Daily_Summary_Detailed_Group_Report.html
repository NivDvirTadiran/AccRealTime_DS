Daily Summary Detailed Group Report

The Daily Summary Detailed Group Report shows detailed information on the data that was accumulated since midnight for a particular group. The Daily Summary information is displayed in a form window.

To create a Daily Summary Detailed Group Report:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select Groups from the Template tab.

  1. Select the Daily Detailed Report template. The Select Group dialog box is displayed.

  2. Select the group for which you want to obtain detailed status information.

Note: You can change the group presented in the report by selecting the current group in the top bar of the window, and then selecting the required group from the Select Group dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

Note: The colors in some fields indicate that the thresholds for those fields have been exceeded. Thresholds are specified in the Aeonix Contact Center Administration.

Statistics

TSF

The TSF is a parameter that measures the level of service given by the group. It is calculated by dividing the number of calls answered before the target ASA by accepted calls.

Note: There are two other ways to calculate the TSF. The first is to divide it by the number of answered calls. The second is to divide it by the sum of answered and abandoned after target ASA calls. The selection between the options is done in the administration and the selected option is applicable to all groups and DNIS numbers.
