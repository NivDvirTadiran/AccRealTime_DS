Brief Group Report

The Brief Group Report shows information on one or more non Email agent groups for the current interval.

To create a Brief Group Report:

  1. Click  (Open Report or Template) in the Control Area.

  2. In the Open dialog box, select Groups from the Template tab.

  3. Select the Status Brief Report template. The Select Multiple Group dialog box is displayed.

  4. Select the groups for which you want to obtain agent group status information, and then click Select.

Note: You can change the groups presented in the report by selecting the current groups in the top bar of the window, and then selecting the required groups from the Select Multiple Group dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

Note: The colors in some fields indicate that the thresholds for those fields have been exceeded. Thresholds are specified in the Aeonix Contact Center Administration.

The following columns are displayed in the Brief Group Status Report:

Group Name

The name of the group.

Group No.

The number assigned to the group.

InB Above T.ASA

The number of Inbound calls, currently in the queue, for which the target Average Speed of Answer was exceeded.

InB Calls in Q

The number of Inbound calls currently in the queue.

InB Max in Q

The longest time of an Inbound call currently waiting in the queue.

OACD in Q

The number of Outbound ACD calls currently in the queue waiting to be answered. The reason for the Outbound ACD to be in the queue is because the reserved agent is unavailable to handle this call.

OACD pend in Q

The number of Outbound ACD calls pending to be called.

No. Agns

The number of agents that are currently logged in.

No. Release

The number of agents in the group that are in the Release state.

No. Idle

The number of agents in the group that are Idle.

No. InACD

The number of agents currently handling Inbound ACD calls.

No. OACD

The number of agents currently handling Outbound ACD calls.

Calls InACD

The number of Inbound VOICE ACD calls handled so far by the group, for the current interval.

Calls OACD

The number of Outbound ACD calls handled so far by the group for the current interval.

TSF

The Target Service Factor. The TSF indicates how well the group is providing the level of service defined for it, via the Target ASA (Average Speed of Answer).

Note: The TSF will be calculated according to the method predefined in the Aeonix Contact Center Administration.

InCalls Abnd

The number of Inbound ACD calls that were abandoned before being serviced by the group in the current interval.

InCalls Intrfl

The number of Inbound ACD calls that were re-routed to another destination, due to call interflow in the current interval.

OACD failed

The number of Outbound ACD calls that have failed.

A Failed outbound ACD call is a call that was successfully generated, but was not handled by an agent. For example, the customer hanging-up while the call is ringing at the agentâ€™s phone.

CHAT Contacts

The number of Inbound CHAT contacts handled so far by the group, for the current interval.

 

Note:

You can adjust the width of the columns by dragging the bottom right hand corner of the window.
