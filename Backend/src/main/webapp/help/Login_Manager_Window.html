Login Manager Window

The Login Status window allows you to monitor the Groups and Agents Groups that are logged into the system. The user can move agents into different groups as needed. In addition, the user can logout either the agent or Group as needed.

To open the Login Status window, click the  icon in the Control Area. The Login Status window appears.

The Login Status window displays both Agents and their respective groups and Groups and their respective agents. The Login Status window is described below:

Agent Login Status

Shows the status of all the agents.

Agents

Lists all agents in use.

Name

The name of each agent in use.

Number

The number of the respective agent in use.

Logged-in-Group

Shows the status of all the Groups in use.

Name

The name of each group in use.

Available groups

Lists all available groups not in use.

Groups Login Status

Shows the status of all the groups.

Groups

Lists all groups in use.

Group Name

The name of each group in use.

Logged in agents

Lists all agents logged into the system.

Name

The name of each agent in use.

Number

The number of respective agent in use.

Available Agents

Lists all available agents not in use.

Name

The name of each agent not in use.

Number

The number of each agent not in use.

Quick Search

Searches for the specified agent or group.

The left arrow button is used to login a new agent or group.

The right arrow button is used to logout an agent or group
