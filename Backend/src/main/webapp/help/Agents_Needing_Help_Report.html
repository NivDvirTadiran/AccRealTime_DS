Agents Needing Help Report

The Agents Needing Help window shows the details of agents in the system who requested help. The number of agents requiring help is always visible in the lower left-hand corner of the Aeonix Web Real-Time screen.

To create an Agents Needing Help Report:

  1. Click  (Open Report or Template) in the Control Area.

  2. In the Open dialog box, select Agents from the Template tab.

  3. Select the Need Help Report template. The Agents Needing Help Report is displayed showing all the agents who request help.

  4. You can save the report by selecting (Save Report As) in the Control Area.

The Agents Needing Help Report contains the following fields:

Column

Description

Agent Name

The name of the agent.

Agent No.

The ID number entered by the agent on login. This number is used to identify the agent in the system.

Agent Exten.

The internal extension that the agent is currently logged in on.

State Time

The length of time that the agent is waiting for help.
