Email Group STI Graph

Note: The  icon denotes that this particular function is not available in Aeonix Contact Center Lite.

The Email Group STI Graph shows the graphical Service Time Interval breakdown for several groups. Each color in the graph represents a different group. The values in the left column indicate the number of calls waiting in the queue for each time interval. The graph presents six intervals (ST1 ... ST6) that represent the time periods specified for the group in the Service Time Intervals field of the Aeonix Contact Center Administration.

To create an Email Group STI Report:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select Groups from the Template tab.

  1. Select the STI Graph template. The Select Multiple Group dialog box is displayed.

  2. Select the groups for which you want to obtain graphical STI information, and then click Select.

Note: You can change the groups presented in the report by selecting the current groups in the top bar of the window, and then selecting the required groups from the Select Multiple Group dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.
