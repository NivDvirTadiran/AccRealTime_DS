Brief Email Group Report

Note: The  icon denotes that this particular function is not available in Aeonix Contact Center Lite.

The Brief Group Report shows information on one or more Email agent groups for the current interval.

To create a Brief Email Group Report:

  1. Click  (Open Report or Template) in the Control Area.

  2. In the Open dialog box, select Groups from the Template tab.

  3. Select the Email Brief Report template. The Select Multiple Email Group dialog box is displayed.

  4. Select the group for which you want to obtain agent group status information, and then click Select.

Note: You can change the groups presented in the report by selecting the current groups in the top bar of the window, and then selecting the required groups from the Select Multiple Email Group dialog box.

  1. You can save the report by selecting  (Save Report As) in the Control Area.

Note: The colors in some fields indicate that the thresholds for those fields have been exceeded. Thresholds are specified in the Aeonix Contact Center Administration.

 

The following columns are displayed in the Brief Email Group Report:

Group Name

The name of the group.

Group No.

The number assigned to the group.

InB Above T.ASA

The number of Inbound email contacts currently in the queue, for which the target Average Speed of Answer was exceeded.

Emails  in Q

The number of Inbound email contacts currently in the queue.

Max Emails In Q

The longest time of an Inbound email contact currently waiting in the queue.

No. Agns

The number of agents which are currently logged in.

No. Release

The number of agents in the group that are in the Release state.

No. Idle

The number of agents in the group that are Idle.

No. InACD

The number of agents currently handling Inbound ACD calls.

Emails Accepted

The number of Inbound ACD email contacts accepted to the group, for the current interval.

Emails Answered

The number of Inbound ACD email contacts answered by the group, for the current interval.

Emails Intfl

The number of Inbound ACD email contacts accepted to the group and interflowed out, for the current interval.

 

Note:

You can adjust the width of the columns by dragging the bottom right hand corner of the window.
