Group Overflow/Interflow Statistics Report

The Group Overflow/Interflow Statistics Report shows detailed information about calls that were directed to the group and calls that were waiting in other groups and were routed to the current group, due to call overflow.

To create a Group Overflow/Interflow Statistics Report:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select Groups from the Template tab.

  1. Select the Overflow-Interflow Report template. The Select Group dialog box is displayed.

  2. Select the group for which you want to obtain detailed status information. The Group Overflow/Interflow Statistics Report is displayed.

Note: You can change the group presented in the report by selecting the current group in the top bar of the window, and then selecting the required group from the Select Group dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

The Group Overflow/Interflow Statistics Report is divided into two main areas; Calls Presented to Group and Treatment of Calls Presented to Group.

The following fields are displayed in the Calls Presented to Group area of the window:

Total Accepted

The sum of the calls that were directed to the group (calls received directly from the CO, transferred to the group or interflowed to the group), and calls waiting in other groups and were routed to the current group due to call overflow.

Direct Calls (DC)

The total number of ACD calls that were directed to the group (i.e., received directly from the CO, transferred to the group or interflowed to the group).

Overflowed In (OvflwIN)

The total number of ACD calls that were waiting in other groups and came in via call overflow.

The following fields are displayed in the Treatment of Calls Presented to Group area of the window:

Answered By Group

The number of calls that were answered in this group. This number includes:

DC Answered By Group

The number of direct calls (as defined in Direct Calls DC) that were routed to this group, and were answered by this group.

OvflwIN Answered By Group

The number of calls routed to this group due to overflow, and were answered by this group.

Answered By Another Group

The number of calls that were answered in another group. This number includes:

DC Answered By Another Group

The number of direct calls (as defined in Direct Calls DC) that were routed to this group, and were answered by another group.

OvflwIN Answered By Another Group

The number of calls that were routed to this group, due to overflow, that were answered by another group.

Interflowed Out

The number of calls that were interflowed out. This number includes:

DC ->Interflowed Out

The number of direct calls (as defined in Direct Calls DC) that were routed to this group, and were interflowed out of this group.

OvflwIN -> Interflowed Out

The number of calls routed to this group due to overflow and were interflowed out.

Abandoned

The number of abandoned calls. This number includes:

DC Abandoned

The number of direct calls (as defined in Direct Calls DC) that were routed to this group and were abandoned.

OvflwIN Abandoned

The number of calls routed to this group due to overflow, and were abandoned.
