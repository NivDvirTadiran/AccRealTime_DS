Group OACD STI

The Group Outbound STI Graph Window is designed to show the distribution of outbound calls waiting (in a group queue) for an agent to become available. Even though this scenario is rare, it is very important to notice it. Each STI shows the number of outbound calls in queue and the amount of time they have been waiting. The longer the calls wait, the more they move to the right of the graph. The STI is determined by the time intervals specified in the Aeonix Contact Center Administration. Each color in the graph represents a different group. The values in the left column indicate the number of calls waiting in the queue for each time interval. The graph presents six intervals (ST1 - ST6) that represent the time periods specified for the group in the Service Time Intervals field of the Aeonix Contact Center Administration.

To create an OACD STI Report:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select Groups from the Template tab.

  1. Select the OACD STI Graph template. The Select Multiple Group dialog box is displayed.

  2. Select the groups for which you want to obtain the status, and then click Select.

Note: You can change the groups presented in the report by selecting the current groups in the top bar of the window, and then selecting the required groups from the Select Multiple Group dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.
