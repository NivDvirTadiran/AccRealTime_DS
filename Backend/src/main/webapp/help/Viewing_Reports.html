Viewing Reports


A report can be saved in the system in one of two lists: the Public report list, and the Private report list. All the authorized supervisors can view public reports. Only the supervisor who created them can view private reports.

To view an existing report:

  1. Select Open from the File menu.

  2. In the dialog box that opens, select the Public or Private check box, or both.

  3. Select the Report Type (i.e., Real-Time, Daily Summary or Hourly).

  4. In the list box, either double-click the report you want to view, or select the report and click OK.

  5. To save a report, select Save from the File menu. To save the report with a new name, select Save As. The report is saved as public or private, according to the selected option.

Note: An asterisk (*) in the title bar of the report indicates that the report structure was changed. This is a reminder that the report needs to be saved for the changes to be written to the database. After the report is saved, the asterisk disappears.
