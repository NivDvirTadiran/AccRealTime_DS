IVR Applications ACD Distribution Graph

Note: The  icon denotes that this particular function is not available in Aeonix Contact Center Lite.

The pie chart displayed in the IVR Applications ACD Distribution Graph shows the distribution of incoming ACD calls received by the IVR application in the current time interval. You can see immediately the percentage of calls that have been Fully Handled, Failed or interflowed in the current interval.

To create an IVR Applications ACD Distribution Graph:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select IVR Applications from the Template tab.

  1. Select the ACD Distribution Graph template. The Select IVR App dialog box is displayed.

  2. Select the IVR application for which you want to obtain graphical call distribution information. The IVR Applications ACD Distribution Graph is displayed.

  1. You can save the report by selecting (Save Report As) in the Control Area.
