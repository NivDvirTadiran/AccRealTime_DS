Report Windows

Note: The  icon denotes that this particular function is not available in Aeonix Contact Center Lite.

This section describes the report windows that you can generate in Aeonix Web Real-Time, for each of the three report types: Real-Time, Daily Summary and Hourly.

For each report window, the following information is presented:

  • The name of the window

  • A brief explanation of the purpose of the window and its usage

  • A list of the columns/fields in the window with a brief explanation of each column/field for reference purposes
