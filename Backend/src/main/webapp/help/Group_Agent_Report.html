Group Agent Report(1 and 2)

The Group Agent Report shows the type of activity in which each agent of a particular group is engaged, and for how long the agent has been engaged in that activity.

Note: The '1' or '2' designates either utilizing 1 or 2 sets of columns.

To create a Group Agent Status Report:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select Groups from the Template tab.

  1. Select the Agent Report template. The Select Group dialog box is displayed.

  2. Select the group for which you want to obtain agent status information. The requested information is displayed.

Note: You can change the group presented in the report by selecting the current group in the top bar of the window, and then selecting the required group from the Select Group dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

The Group Agent Status Report contains the following columns for each agent in the group:

Agent Name

The name of the agent.

Agent Number

The ID number entered by the agent on login. This number is used to identify the agent in the system.

Agent Extension

The internal extension on which the agent is currently logged in.

Agent State

The agent’s state. This can be one of the following:

Idle

No calls are being handled and the agent is free to receive calls.

Release

The state of an agent when the agent activates the Release button. In this state the agent is still part of the ACD but is not available to receive new ACD calls. The station can still receive incoming non-ACD calls in this state.

Ring

An ACD call is ringing at an agent’s extension.

Wrap Up

The agent has completed a call and is now in the wrap-up state.

ACD

The agent is currently handling an ACD call.

Split

The agent is performing a task in an ACD group other than the one displayed. A Split is followed by a number, indicating the ACD group in which the agent is active. The display must be switched to this group to determine the exact state of the agent.

Busy

The agent station is busy.

Inc

The agent answered an incoming, non-ACD call.

Out

The agent is handling an outgoing, non-ACD call.

Out+Rls

The agent makes an outgoing outside line call, while the station is in the Release state.

Inc+Rls

The agent is talking on an incoming call (i.e., non-ACD call), while the station is in Release state.

OACD

The agent is handling an outbound ACD call.

O-Rsrv

The agent is reserved to be connected to an Outbound ACD call.

Bsy+Rls

The agent is ringing an incoming non-ACD call or dialing an outgoing non-ACD call, while the station is in Release state.

State Time

The time the agent station is in the state indicated in the State column.

Notes: 1. When the agent completes a non-ACD activity while in an Release state, the State Time continues to count the Release time from the time the agent first entered the Release state.

2. When in Idle state, the agent’s extension becomes busy with a non-ACD call and then returns to Idle state, the State time continues to count the Idle time from the last ACD call the agent completed.

Release code

This column shows, for every released agent, the name of the release code selected by the agent. Note that there are two system defined release codes. The first is ‘No Code’ and the second is ‘Forced Released’.

Inbound ACD Calls

This column shows the number of incoming ACD Voice calls that the agent is currently handling.

Outbound ACD Calls

This column shows the number of outbound ACD Voice calls that the agent is currently handling.

NON-ACD Calls

This column shows the number of incoming non-ACD Voice calls that the agent is currently handling (incoming as well as outgoing).

Chat Contacts

This column shows the number of incoming ACD chat contacts that the agent is currently handling.

Email Contacts

This column shows the number of incoming ACD email contacts that the agent is currently handling.

Hold

The number of calls that are put on hold.

ANI

The ANI value that arrived with the current call of the agent.

DNIS

The DNIS value that arrived with the current call of the agent.

The different colors used in the window fields indicate different station states. The state colors are specified in the Colors option of the Aeonix Contact Center Administration.

Note:

You can adjust the width of the columns by dragging the bottom right hand corner of the window.
