Brief Email DNIS Report

Note: The  icon denotes that this particular function is not available in Aeonix Contact Center Lite.

The Brief DNIS Report shows information about one or more Email DNIS numbers for the current interval.

To create a Brief Email DNIS Report:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select DNIS from the Template tab.

  2. Select the Email Brief Report template. The Select Multiple Voice DNIS dialog box is displayed.

  3. Select the DNIS's for which you want to obtain agent group status information, and then click Select.

Note: You can change the DNIS's presented in the report by selecting the current DNIS's in the top bar of the window, and then selecting the required DNIS's from the Select Multiple Voice DNIS dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

The following columns are displayed in the Brief Email DNIS Report:

DNIS Name

The name of the DNIS.

DNIS No.

The DNIS number.

Above T.ASA

The number of email contacts, currently waiting in the queue, for which the target Average Speed of Answer (TASA) was exceeded.

Calls in Q

The number of email contacts currently waiting in the queue.

Max in Q

The longest time of a DNIS email contact currently waiting in the queue for the DNIS.

Calls ACD

The number of ACD DNIS email contacts that arrived in the current interval.

Calls Ansd

The number of ACD DNIS email contacts answered for the current interval.

Avg. Q Time

The average time a DNIS email contact waited in the queue, for the current interval.

Max Q Time

The maximum time a DNIS email contact waited in the queue, for the current interval.

 

Note:

You can adjust the width of the columns by dragging the bottom right hand corner of the window.
