Agent Distribution Graph Window

The Agent Distribution Graph is a pie chart displaying the current number of agents in each state for the selected group.

To create an Agent Distribution Graph Report:

  1. Click  (Open Report or Template) in the Control Area.

  2. In the Open dialog box, select Groups from the Template tab.

  3. Select the Agents Distribution Graph template. The Select Group dialog box is displayed.

  4. Select the group for which you want to obtain graphical call distribution information. The requested information is displayed.

Note: You can change the group presented in the report by selecting the current group in the top bar of the window, and then selecting the required group from the Select Group dialog box.

  1. You can save the report by selecting  (Save Report As) in the Control Area.
