Brief DNIS Report

The Brief DNIS Report shows information about one or more DNIS numbers for the current interval.

To create a Brief DNIS Status window:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select DNIS from the Template tab.

  2. Select the Brief Report template. The Select Multiple Voice DNIS dialog box is displayed.

  3. Select the DNIS's for which you want to obtain agent group status information, and then click Select.

Note: You can change the DNIS's presented in the report by selecting the current DNIS's in the top bar of the window, and then selecting the required DNIS's from the Select Multiple Voice DNIS dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

The following columns are displayed in the Brief DNIS Report:

DNIS Name

The name of the DNIS.

DNIS No.

The DNIS number.

Above T.ASA

The number of calls, currently waiting in the queue, for which the target Average Speed of Answer (TASA) was exceeded.

Calls in Q

The number of calls currently waiting in the queue.

Max in Q

The longest time of a DNIS call currently waiting in the queue for the DNIS.

Calls ACD

The number of ACD DNIS calls that arrived in the current interval.

Calls Ansd

The number of ACD DNIS calls answered for the current interval.

A DNIS answered call is a call that was answered at least once. As opposed to a group, a DNIS retains its identity until it is disconnected. Thus, a DNIS call can be answered more than once during its lifetime. The answer time is measured by the first time it is answered.

Calls Abnd

The number of DNIS calls that were abandoned before being answered.

TSF

The Target Service Factor. The TSF indicates how well the DNIS is providing the level of service defined for it, via the Target ASA (Average Speed of Answer).

Note: The TSF will be calculated according to the method predefined in the Aeonix Contact Center Administration.

Avg. Wait Abnd

The average waiting time that abandoned DNIS calls have waited before hanging up, for the current interval.

Max Wait Abnd

The maximum time that abandoned DNIS calls waited before hanging up, for the current interval.

Avg. Q Time

The average time DNIS calls wait in the queue, for the current interval.

Max Q Time

The maximum time DNIS calls wait in the queue, for the current interval.

 

Note:

You can adjust the width of the columns by dragging the bottom right hand corner of the window.
