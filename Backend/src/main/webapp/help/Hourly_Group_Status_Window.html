Hourly Group Status Window

The Hourly Group Status window shows information about one agent group based on an Hourly Report interval.

Note: Statistics for hourly reports are not accumulated in real-time.

To create an Hourly Group Status window:

  1. Click  (Open Report or Template) in the Control Area.

  1. In the Open dialog box, select Groups from the Template tab.

  1. Select the Hourly Report template. The Select Group dialog box is displayed.

  2. Select the group for which you want to obtain agent group status information.

Note: You can change the group presented in the report by selecting the current group in the top bar of the window, and then selecting the required group from the Select Group dialog box.

  1. You can save the report by selecting (Save Report As) in the Control Area.

The following columns are displayed in the Hourly Group Status window:

Field

Description

Interval Start At

The Historical Report interval start time.

Total In

The total number of accepted calls during the specified interval.

Calls Ansd

The number of answered calls during the specified interval.

Calls Abnd

The number of abandoned calls during the specified interval.

Avg. Talk

The average talk time of an ACD call.

Avg. Abnd

The average time an abandoned call waited in the queue.

Avg. Wrap

The average time the agent spent on wrap-up after a call.

 

Note: The Group Hourly Report template is rigid – neither the columns’ sequence nor their width may be changed.
